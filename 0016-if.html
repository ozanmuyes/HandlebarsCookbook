Handlebars: Built-in block helper - #if

Provide simple if/else rendering logic.

This is a handlebars.js extension, mustache do not support this.

Samples

True!

lightncandy

Used option: FLAG_ELSE

Data:
array(
  "foo" => true
)
Template:
{{#if foo}}YES{{else}}no{{/if}}
Result:
YES
Source Code 
require('./vendor/autoload.php');
use LightnCandy\LightnCandy;
$template = "{{#if foo}}YES{{else}}no{{/if}}";

$php = LightnCandy::compile($template, array(
  "flags" => LightnCandy::FLAG_ELSE
));
$render = LightnCandy::prepare($php);
$data = array(
  "foo" => true
);
echo $render($data);
handlebars.js
Data:
{
  "foo": true

var render = Handlebars.compile(template);

  "foo": true
};
console.log(render(data));
